# 访问分析与近实时视图

访问分析是可选功能。它需要 `ANALYTICS` 绑定，以及能够查询 Analytics Engine 的 Cloudflare 账户凭据。配置的数据集名称必须与绑定的数据集一致。

## Sink 记录的内容

成功的链接访问可用于生成计数器、时间序列视图、指标、热力图、最近事件和位置数据。可用过滤条件包括链接、时间范围、国家或地区、浏览器、操作系统、设备和来源网站。Sink 使用 Analytics Engine 的采样结果，因此大流量下的数值可能是估算值，而非逐笔事务记录。

检测到的机器人流量可以同时从访问统计和[点击 Webhook](/zh-CN/configuration/webhooks)中排除。

## 近实时仪表盘

近实时视图有意采用模拟实时的方式。仪表盘每 10 秒轮询一次访问分析，发现新的访问事件，并在客户端以大约每秒一个事件的速度回放队列中的事件。它**不使用** Server-Sent Events 或 WebSocket。

队列有容量上限。暂停会停止轮询、事件回放和 WebGL 运动；页面隐藏后也会暂停回放，直到页面再次可见。近实时视图应视为运维可视化，而不是无损事件流。

## 导出

使用仪表盘或已认证的统计导出路由，可以将筛选后的访问分析下载为 CSV。导出包含 Slug、URL、访客数、访问量和来源网站等汇总列。链接 JSON 导出是另一项功能，详见[导入/导出](./import-export)。

凭据、数据集、列表限制和机器人过滤设置详见[配置参考](/zh-CN/configuration/#推荐配置)。
